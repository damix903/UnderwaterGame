# Deep Dive

## 概要
リソース管理を行いながら下を目指していく2Dアクションゲームです。

## ゲーム内容
- 攻撃にはHPが必要で、敵を撃破することでHPを回復することができる
- 地面に着地しないで敵を連続で撃破するとコンボを繋げることができる
- コンボ数に応じて敵撃破時のHP回復量が変化

## 操作方法
- A/D : 横方向に移動
- マウス移動 : 弾を発射する方向を決める
- 左クリック : 弾を発射

## 制作意図
リソースを消費するリスクとリターンの判断を継続的にプレイヤーに迫ることをコアの体験として制作しました。なるべく判断することに思考を集中させたいため操作はシンプルにしましたが、攻撃という1つの行動に複数の意味を持たせることで奥深いゲーム体験になるよう工夫しました。

例えばリソースの消費と回復だけでなく、攻撃の際に通常の横移動よりも大きく動くようにすることで移動も兼任されています。攻撃を移動手段として機能させることで「攻撃するほど危険だが自由度が増す」という状態を作り、より判断を迫ることを目指しました。

## 使用技術
- Unity
- C#

## 現状の課題
- 深度の変化によるゲーム体験の変化がない
- 上手いプレイに対する報酬の不足
- ステージ生成方法の単調さ
- 判断を変えるような敵やギミックの不足
